#player example

player_core is an abstraction package — it has no runnable example on its own.
See [`audio_player/example/`](../../audio_player/example/) for
a runnable Flutter app demonstrating the wrapper end-to-end (single-track,
playlist, multi-scope, observer flows).

The [`main.dart`](main.dart) here is a doc-only snippet illustrating the
API shape; pair it with any concrete impl.
